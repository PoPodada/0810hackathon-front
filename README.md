# Hicoder&IE 合同ハッカソン チーム0

## フロントエンド環境構築手順


1. パッケージのインストール
```
pnpm install
```

2. 起動
```
pnpm run start
```
pnpm run start によって出力されたQRコードを読み取るとモバイルにて動作確認をすることができます

エミューレーターを設定することによってpcでandroidとiosの動作確認をすることもできます
android: https://docs.expo.dev/workflow/android-studio-emulator/

ios: https://docs.expo.dev/workflow/ios-simulator/
